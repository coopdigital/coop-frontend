# InputCheckboxGroup

Intro to InputCheckboxGroup. Allows for grouping of InputCheckbox components under a fieldset with legend and optional hint text

## Usage

Use it in a form to group related checkboxes in a `<fieldset>`

## Default InputCheckboxGroup example

A test cnavas to show the component and it's underlying code.
A canvas can include stories or just plain components like the one below.
