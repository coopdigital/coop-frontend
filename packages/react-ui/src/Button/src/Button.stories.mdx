# Buttons

Buttons are interactive elements that people can use to take an action.

[Go to buttons page on the Experience Library](https://pioneer-design-system.herokuapp.com/elements/buttons.html).

## Installation
Install via NPM:
```bash
$ npm install @coopdigital/foundations-buttons --save
```

## Usage

### CSS
```css
@import "node_modules/@coopdigital/foundations-colors/dist/buttons.css";
```

### PostCSS
```css
@import "node_modules/@coopdigital/foundations-colors/src/buttons.pcss";
```

### PostCSS Import
```css
@import "@coopdigital/foundations-buttons";
```

### Button sizes

Buttons have 3 sizes:

- default, no variant prop required
- for a small button, add the value `small` to your `variant` prop on the component

<Story
  name="Base/Buttons sizes"
  args={{
    children: "Label",
    handleClick: testMyClick,
  }}
>
  
  
  <Button handleClick={testMyClick} size="small">Small</Button>
</Story>
