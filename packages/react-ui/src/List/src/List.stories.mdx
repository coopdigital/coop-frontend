# List
Lists are an effective way of breaking up content â€” they make it easier for users to scan and find information. Use bullet points to break up a list of items.

## Installation
Lists come with the typography package. Install this by running the command in your project.

`npm install @coopdigital/foundations-typography`

Then import the CSS file into your project's main CSS file.

`@import "@coopdigital/foundations-typography";`

## Usage

### HTML
The HTML snippets for lists can be found in the Canvas tab under HTML in the addons panel.

### Bullet list
<Canvas>
  <List type="bullet" items={items} />
</Canvas>

### Numbered list
<Canvas>
  <List type="numbered" items={items} />
</Canvas>

### CSS
Utility classes to help you style lists.

`.coop-u-list-bare` Removes the bullet points or numbers and the left hand padding of the list.

`.coop-u-list-inline` Will put the list into a horizontal row.

### List bare example
