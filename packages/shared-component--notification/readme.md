# Notification

The Notification component can be used as either an alert or status notification.

The contentful properties available are:

- Name
- Header
- Body text
- Link (reference to another link)
- Type of notification (alert or notification)

#
## Config options

```TypeScript
{
  "id": String - required
}
```
